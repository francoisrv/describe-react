# Selectors

You can use of these selectors

- `element` or `elements`
- `child` or `children`
- `parent` or `parents`
- `sibling` or `siblings`

Each selector can be accompanied by a modifer, ie `first={5} children`

Each selector can be also accompanied by a `which` modifier, ie `first={5} children which have text="abc"`
